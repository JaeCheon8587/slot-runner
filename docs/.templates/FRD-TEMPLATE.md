# FRD-TEMPLATE — Legacy Redirect

> ⚠ **LEGACY TEMPLATE** — 0.3 신규 문서 작성에는 사용하지 않는다.

0.3 문서 체계의 FRD는 전역 `FRD/` 문서가 아니라 App별 `docs/{App}/FRD/` 하위 문서이다.

신규 기능 상세 문서 작성 시 아래 템플릿을 사용한다.

- 원본: `.templates/App/FRD/APP-FRD-001-TEMPLATE.md`
- 결과: `docs/{App}/FRD/{App}-FRD-{NNN}.md`
- 기능 카탈로그: `docs/{App}/{App}-FC.md`

작성 순서는 [`DOCUMENT_GUIDE §2`](DOCUMENT_GUIDE.md#2-작성-순서) 의 "신규 기능 추가" 절을 따른다.
